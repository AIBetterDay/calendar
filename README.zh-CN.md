# 待办

把脑子里的事情变成可执行任务，安排到日历里，再让 Better 帮你持续推进。

[English](./README.md)

## 它解决什么问题

待办适合“一个想法刚刚变成行动”的时刻。你可以用自然语言添加任务，补上日期或提醒，然后在专注列表或日历视图里复盘和安排。

## 你可以用它做什么

- 记录标题、备注、优先级、标签、子任务、截止日期、重复规则和提醒。
- 用日视图、周视图、月视图按时间规划任务。
- 让 Better 查找逾期任务、批量改期、完成 errands 或整理列表。
- 在同一个工作区里看清状态、优先级、提醒和上下文。

## 可以这样问

- “提醒我下周五上午更新护照。”
- “列出逾期待办，把低优先级的移到下周。”
- “把 errands 标签下的事项全部标记完成。”

## 界面预览

产品截图和短演示会放在 `assets/store/`。目前先不放占位图；等真实产品图片加入后，再更新商店详情页。

## 对话里的交互卡片

当这个 connector 在 Better 对话里返回结构化结果时，请从 `ui/src/cards/` 渲染产品化卡片，不要直接展示裸 JSON 或大段纯文本。卡片资源和样式应该跟 UI 代码放在一起。

## 隐私

待办数据存储在你设备上的 Better 里。它不会访问外部服务，也只使用自己的 connector 存储空间。
